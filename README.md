# ISPM-15 Compliant SBC-Based Multi DS18B20 Temperature Monitoring System

Raspberry Pi tabanlı, çoklu DS18B20 sensör kullanan ve ISPM-15 standartlarına uygun
sıcaklık izleme ve veri kayıt sistemi.

## Project Scope
- SBC tabanlı sıcaklık ölçümü
- 15 adet DS18B20 (1-Wire)
- 1 dakikalık ölçüm periyodu
- Veri kaydı ve raporlama
- ISPM-15 uyumluluğu

## System Status
- Prototype completed
- Field deployment successful
- Approved through Ministry of Agriculture testing process
